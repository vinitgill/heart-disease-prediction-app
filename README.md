# ❤️ AI Heart Disease Prediction System
### Developed by Vinit Gill

A production-quality machine learning application built with **Streamlit**, **Scikit-Learn**, and **XGBoost**. This system predicts the risk of cardiovascular disease based on clinical parameters and features a unique **Neobrutalism UI** with **Anti-Gravity physics-inspired animations**.

## 🚀 Features
- **Predictive Engine:** Trained on clinical data using Random Forest (91% ROC-AUC).
- **Multi-Tab UI:** 
    - 🏠 **Predictor:** Real-time risk analysis.
    - 📊 **Analytics:** Model performance and feature importance.
    - 📖 **Health Guide:** Tips for a healthy heart.
- **Visuals:** High-contrast Neobrutalism design with dynamic CSS animations.
- **Deployment Ready:** Configured for Streamlit Cloud and GitHub.

## 🛠️ Tech Stack
- **Languages:** Python (Pandas, Numpy, Scikit-Learn, XGBoost)
- **Framework:** Streamlit
- **Design:** CSS (Neobrutalism Theme)
- **Deployment:** Streamlit Cloud

## 📦 Installation & Usage
1. **Clone the repository:**
   ```bash
   git clone https://github.com/YOUR_USERNAME/heart-disease-predictor.git
   cd heart-disease-predictor
   ```
2. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```
3. **Run the App:**
   ```bash
   streamlit run app.py
   ```

## 🧠 Training
The model was trained using the provided `train.ipynb` notebook. It evaluates Logistic Regression, Random Forest, and XGBoost, selecting the best model based on ROC-AUC scores.
